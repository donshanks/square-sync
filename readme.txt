Square Sync lists squareup products from the Connect Squareup API

== Description ==

Square Sync allows you to list you Squareup products on your wordpress page/post(s) via shortcode. The products are listed from the Connect Squareup API and link to the Square Market. As of this version, the Square Market's payment processing is only available to US customers.

== Installation ==

1. Upload the plugin to the '/wp-content/plugins/' directory;
3. Activate the plugin through the 'Plugins' menu in WordPress;
4. Enter your API key in the Square settings page and other options desired;
6. Place '[squaresync]' in your page or post;

== Frequently Asked Questions ==

= How do I get an API key? =

Login to you squareup account at connect.squareup.com and click on 'APPS'. Click on the 'New App' button on the right of screen and enter an app name. You will know have a personal token.
